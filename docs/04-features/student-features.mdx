# Student Features

**Version:** 1.0
**Last Updated:** May 2026

---

## F-ST-001 — Subject-wise Material Library

**User:** Student
**Priority:** P0

**What it does**
Central place where students browse all study materials uploaded by teachers, organized by subject.

**Behaviour**
- Home screen has "Library" tab
- Materials grouped by subject — Maths, Science, Gujarati, etc.
- Each material shows: title, subject, chapter, teacher name, upload date, file size
- Tapping opens the file in-app viewer (F-ST-002)
- New materials since last visit marked with a "New" badge
- Search bar to find material by title or subject
- Loads from local cache when offline, shows "Last updated" timestamp

---

## F-ST-002 — PDF Viewer In-App

**User:** Student
**Priority:** P0

**What it does**
Opens PDF and image files directly inside the app without needing a third-party app.

**Behaviour**
- Supports PDF and JPG/PNG
- Pinch to zoom
- Swipe to next page on PDF
- Download button visible while viewing
- Works fully offline for downloaded files
- For non-downloaded files: shows loading indicator, falls back to "Download to read offline" if no internet

## F-ST-003 — Offline Note Download

**User:** Student
**Priority:** P0

**What it does**
Student downloads materials to phone storage and reads them with no internet.

**Behaviour**
- Download button on every material in library
- Download progress bar shown
- Downloaded files stored in app's private storage — not accessible from file manager by other apps
- Airplane mode test: downloaded files open fully offline
- Storage full: show clear error "Storage full. Delete some files to download more"
- Download fails midway: show retry button
- File updated by teacher: show "Updated version available" badge on downloaded file

**Limits**
- Maximum 500MB total offline storage per student
- Warning shown at 400MB
- Individual file limit: 25MB

## F-ST-007 — Pending Assignments List

**User:** Student
**Priority:** P0

**What it does**
Shows all assignments the student needs to complete.

**Behaviour**
- Separate tabs: Pending, Submitted, All
- Each assignment shows: subject, title, due date, days remaining
- Overdue assignments shown in red at top of list
- Due today shown in amber
- Tapping assignment opens detail with description and submission option
- Updates from cache when offline

## F-ST-008 — Assignment Submission via Photo Upload

**User:** Student
**Priority:** P0

**What it does**
Student submits handwritten or digital work by uploading a photo or PDF.

**Behaviour**
- Two options: Take Photo or Upload from Gallery
- Multiple photos allowed per submission (max 5 images or 1 PDF)
- Max file size: 10MB total per submission
- Preview before final submit
- Submission timestamp recorded server-side
- Cannot resubmit after teacher has graded — only before grading
- Confirmation screen shown after successful submission

---

## F-ST-009 — Offline Assignment Queue

**User:** Student
**Priority:** P0

**What it does**
If student submits when offline, submission is queued and auto-uploaded when internet returns.

**Behaviour**
- Submission saved locally with "Pending Upload" status
- Banner on home screen: "1 submission waiting to upload"
- Auto-uploads in background when internet detected
- If deadline passes while still queued: submission marked as late, teacher can see it was queued before deadline
- Student notified when upload completes

## F-ST-014 — Gujarati Language Support

**User:** Student
**Priority:** P0

**What it does**
Entire app UI displayed in Gujarati by default.

**Behaviour**
- All labels, buttons, error messages, notifications in Gujarati
- Gujarati font loads as part of app — does not require internet
- Numbers shown in standard numerals (not Gujarati script numerals) for clarity
- Language setting: Gujarati default, Hindi and English available in settings
- Teacher-uploaded content language is not changed — only UI elements
